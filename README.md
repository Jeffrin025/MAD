1.Install Node.js
React Native requires Node.js. You can download and install it from Node.js official website. Ensure you have the latest LTS version.
2. Install React Native CLI
npm install -g react-native-cli
3. Set Up Android Development Environment
4. Create a New React Native Project
npx react-native init MyProject
5. Run Your React Native App
npx react-native run-android
